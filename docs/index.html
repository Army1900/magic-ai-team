Model-driven Team OS CLI

把“单 Agent”升级成
可治理的 AI 团队系统

OpenTeam 不是另一个跑命令的外壳，而是一层 Team OS：从访谈建模、团队拓扑设计、质量门禁、风险审查，到一键导出 OpenCode / Claude / OpenClaw / Codex 等目标框架，形成可落地、可审计、可迭代的团队交付链路。

查看快速开始 阅读源码
upAI discovery + team topology
exportquality gate + policy + compatibility
handoffTEAM_BRIEF + START_PROMPT
startlauncher check + run/monitor

为什么它和普通 Agent 工具不一样

团队级编排，不是单体 Prompt

支持多 Agent 角色、输入输出契约、协作流和资源绑定，贴近真实工程团队分工。

门禁前置，默认更安全

高风险资源默认阻断；quality / policy / compatibility / self-check 串联，避免“导出了才发现不可用”。

可解释的执行轨迹

phase timeline、summary 卡片、Fix now 命令、history/worklog 全链路留痕，方便复盘和审计。

30 秒体验

npm install
npm run build
npm link

openteam
# 默认进入 go 流程: up -> export -> handoff -> start

openteam go --project D:\Projects\my-app --target opencode --view advanced
openteam history --limit 20
openteam viewer --project D:\Projects\my-app
