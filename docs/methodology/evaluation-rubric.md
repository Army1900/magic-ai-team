# Evaluation Rubric

## Metrics
- Task success rate
- Latency (p95)
- Cost per run
- Policy compliance
- Human intervention rate

## Scoring Guidance
- Overall score balances quality, reliability, and efficiency.
- Any policy failure is a release blocker.

## Decision Thresholds
- Promote when score improves and no critical policy failures exist.
- Roll back when reliability regresses.
