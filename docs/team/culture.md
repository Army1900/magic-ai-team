# Team Culture

## Purpose
- Build reliable agent teams that improve user outcomes with measurable quality and cost control.

## Principles
- User goal first.
- Prefer clarity over cleverness.
- Make tradeoffs explicit.
- Default to reversible changes.
- Keep a clear audit trail for major decisions.

## Quality Bar
- No silent failures.
- Every production-facing change must be observable.
- Policy violations block execution unless explicitly overridden.
